# Tic-Tac-Toe

A minimal Tic-Tac-Toe game built with React.

## Features

- Two-player turn-based gameplay (X and O)
- Automatic winner detection
- Draw detection when all squares are filled
- Reset button to start a new game

## Getting Started

```bash
npm install
npm run dev
```

## How to Play

1. X always goes first
2. Click any empty square to place your mark
3. First player to get 3 in a row (horizontal, vertical, or diagonal) wins
4. If all 9 squares are filled with no winner, it's a draw
5. Hit **Reset** to play again
